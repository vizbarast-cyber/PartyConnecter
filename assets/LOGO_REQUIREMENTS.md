# Logo File Requirements

## Current Status
The app is configured to use the PartyConnect logo with dark purple theme.

## Files Needed

1. **icon.png** - 1024x1024px
   - Main app icon
   - Used in: App stores, loading screen, app drawer

2. **adaptive-icon.png** - 1024x1024px  
   - Android adaptive icon
   - Same design as icon.png

3. **splash.png** - 2048x2048px
   - Splash screen image
   - Logo centered on dark background

4. **favicon.png** - 512x512px
   - Web favicon

## Design Specs

**Theme Colors:**
- Primary: #9D4EDD (Purple)
- Background: #0A0A0F (Dark)
- Light Purple: #C77DFF
- Accent: #FF6B9D (Pink)

**Logo Design:**
- Party popper with confetti
- "PartyConnect" text
- Dark purple theme colors
- Transparent or dark background
